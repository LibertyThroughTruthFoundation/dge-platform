## FAMILY OFFICE CONSIDERATIONS

### Sensitive Documents Strategy

**Regardless of which tier you use, protect sensitive documents:**

**High-Security Documents:**
- Legal documents (wills, deeds, passports)
- Financial records (tax returns, bank statements)
- Personal identification
- Family medical records

**Storage Strategy:**
1. **Encrypt locally** before uploading to any cloud (use VeraCrypt or similar)
2. **Store encrypted containers** in cloud storage
3. **Keep encryption keys** separate (password manager or offline)
4. **Physical backups** on external hard drive stored securely at home

**Lower-Security Documents:**
- SCCC operational documents
- Project files (NationOS, GodManMarkets)
- Content creation drafts
- General family photos

**Storage Strategy:**
- Can be stored in any of the cloud solutions
- Still benefit from platform encryption (Google, Proton, NextCloud all encrypt in transit)

## APPENDIX: Quick Command Reference

### Check Google Drive Space:
```bash
rclone about manus_google_drive: --config /home/ubuntu/.gdrive-rclone.ini
```

### Sync Google Drive to NextCloud:
```bash
rclone sync "manus_google_drive:/Sovereign Command Center" \
  "nextcloud:/Bryan_SCCC/" \
  --config /home/ubuntu/.gdrive-rclone.ini
```

### List All Files in SCCC:
```bash
rclone lsf "manus_google_drive:/Sovereign Command Center/" \
  --recursive --config /home/ubuntu/.gdrive-rclone.ini
```

### Backup SCCC Locally:
```bash
rclone copy "manus_google_drive:/Sovereign Command Center" \
  /home/ubuntu/SCCC_Backup/ \
  --config /home/ubuntu/.gdrive-rclone.ini
```
