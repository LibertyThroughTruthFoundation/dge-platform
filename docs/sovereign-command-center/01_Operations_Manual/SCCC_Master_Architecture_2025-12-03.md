### Setup Steps

#### Phase 1: Initial Configuration

1. **Access Your NextCloud Instance**

2. **Create Folder Structure**
   ```
   Family Command Center/
   ```

   - Download from nextcloud.com/install
   - Configure selective sync (choose which folders to sync locally)
   - Set up automatic synchronization

4. **Install NextCloud Mobile App**
   - iOS: App Store
   - Android: Google Play
   - Enable automatic photo upload
   - Configure offline access for key folders

---

#### Phase 2: Integration with Google Drive

**Option A: Manual Sync (Immediate)**
- Regularly copy key documents from Google Drive to NextCloud
- Use rclone for automated copying (set up cron job)

**Option B: NextCloud External Storage (Advanced)**
- Configure Google Drive as external storage in NextCloud
- Requires NextCloud admin access and Google API setup
- Provides unified view of both repositories

**Recommended Command (rclone):**
```bash
# Sync SCCC folder from Google Drive to NextCloud
rclone sync "manus_google_drive:/Sovereign Command Center" \
  "nextcloud:/Bryan_SCCC/" \
  --config /path/to/rclone.ini
```
