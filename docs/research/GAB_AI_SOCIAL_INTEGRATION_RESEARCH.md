## GAB SOCIAL API - CONFIRMED CAPABILITIES

### ‚úÖ **GAB SOCIAL API EXISTS AND SUPPORTS POSTING!**

**Source:** https://gavkaz.github.io/Gab-API-Wrapper/

---

### **REQUIREMENTS:**

1. ‚úÖ **Gab Pro subscription required** (to access API)
2. ‚úÖ **Developer application** (create in User ‚Üí Settings)
3. ‚úÖ **OAuth2 authentication** (get auth token)
4. ‚úÖ **API scopes** (permissions):
   - `read` - Read user data, feeds, groups
   - `write-posts` - Create posts and media
   - `engage-post` - Upvote, downvote, repost
   - `engage-user` - Follow, unfollow
   - `notifications` - Access notifications

---

### **KEY API METHODS:**

#### **POSTING:**
- ‚úÖ **`createPost(options)`** - Create new post
  - `body` - Post text content
  - `media_attachments[]` - Up to 4 media files
  - `gif` - Giphy gif ID
  - `reply_to` - Reply to post ID
  - `is_quote` - Quote post
  - `nsfw` - Mark as NSFW
  - `group` - Post to group
  - `topic` - Post topic
  - `poll` - Create poll with options

- ‚úÖ **`createMediaAttachment(file)`** - Upload media
  - Returns media ID for use in createPost

#### **ENGAGEMENT:**
- ‚úÖ **`upvote(postId)`** - Upvote post
- ‚úÖ **`downvote(postId)`** - Downvote post
- ‚úÖ **`repost(postId)`** - Repost
- ‚úÖ **`removeUpvote(postId)`** - Remove upvote
- ‚úÖ **`removeDownvote(postId)`** - Remove downvote
- ‚úÖ **`removeRepost(postId)`** - Remove repost

#### **SOCIAL:**
- ‚úÖ **`follow(userId)`** - Follow user
- ‚úÖ **`unfollow(userId)`** - Unfollow user
- ‚úÖ **`followers(username)`** - Get followers
- ‚úÖ **`following(username)`** - Get following

#### **READING:**
- ‚úÖ **`me()`** - Get own account info
- ‚úÖ **`user(username)`** - Get user profile
- ‚úÖ **`userFeed(username)`** - Get user's posts
- ‚úÖ **`mainFeed()`** - Get main feed
- ‚úÖ **`popularFeed()`** - Get popular feed
- ‚úÖ **`postDetails(postId)`** - Get post details
- ‚úÖ **`notifications()`** - Get notifications

#### **GROUPS:**
- ‚úÖ **`groupDetails(groupId)`** - Get group info
- ‚úÖ **`groupUsers(groupId)`** - Get group members
- ‚úÖ **`groupModerationLogs(groupId)`** - Get mod logs
- ‚úÖ **`popularGroups()`** - Get popular groups

---

### **POSTING LIMITS:**

**From Gab Help documentation:**
- **1,500 posts per day** (max)
- **Hourly, semi-hourly, and minute-by-minute limits** (not specified)
